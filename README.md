# ⚡ Dotfiles

A minimalist collection of configuration files to synchronize my development environment across machines.

---

## 📂 Repository Contents

| File | Description |
| :--- | :--- |
| `.vimrc` | Vim editor settings, keybindings, and UI tweaks. |
| `.zshrc` | Main Zsh configuration, theme loading, and environment variables. |
| `aliases.zsh` | Custom command shortcuts and automation scripts. |
| `iterm2-preferences` | iterm2 profile preferences. |
---

### Iterm2 profile import

1. Open the Profiles Manager
Launch iTerm2.

Go to Settings (Cmd + ,).

Click the Profiles tab at the top.

2. Import the JSON
Look at the bottom left of the Profiles window for the Other Actions button (usually represented by a ... or a gear icon, depending on your version).

Select Import JSON... from the dropdown menu.

Browse to the location of your iterm2-preferences file and click Open.
